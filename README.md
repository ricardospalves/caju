# Caju

## Instalação e uso

Clone o repositório:

```sh
git clone git@github.com:ricardospalves/caju.git
```

Acesse o projeto:

```sh
cd caju
```

Instale as dependências:

```sh
npm install
```

Rode o servidor:

```sh
npm run init:db
```

Rode a aplicação web:

```sh
npm run dev
```

Acesse a aplicação: http://localhost:5173/

Acesse o JSON Web Server: http://localhost:3000/
